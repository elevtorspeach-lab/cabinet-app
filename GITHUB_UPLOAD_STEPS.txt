GitHub quick upload (from project root):

Current local repository:
   branch : main
   remote : https://github.com/elevtorspeach-lab/cabinet-app.git

Standard publish flow:
   git add .
   git commit -m "Update cabinet app"
   git push origin main

If you want to connect this project to a different GitHub repository:
   git remote remove origin
   git remote add origin https://github.com/<YOUR_USER>/<YOUR_REPO>.git
   git push -u origin main

Useful checks before push:
   git status
   git remote -v
   git branch --show-current

Important:
   - large generated files and node_modules are ignored by .gitignore
   - web deployment uses .github/workflows/deploy-web.yml
   - desktop builds use .github/workflows/build-mac.yml and build-windows.yml

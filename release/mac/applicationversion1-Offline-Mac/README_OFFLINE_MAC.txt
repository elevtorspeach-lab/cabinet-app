applicationversion1 - Version Mac Offline

Utilisation:
1) Dézipper le dossier.
2) Double-cliquer sur LANCER_OFFLINE.command
3) Si macOS bloque l'ouverture:
   - Clic droit > Ouvrir > Ouvrir

Important:
- L'application fonctionne sans connexion internet.
- Les données restent sur ce Mac (stockage local du navigateur utilisé).
- Import/Export Excel fonctionne hors connexion (libs locales incluses).
- Utilisez toujours le même navigateur pour garder la même base locale.
